# KML (Keyboard Macro Language) Specification v1.0

## 1. コンセプト
KMLは、MIDIやMML (Music Macro Language) の思想をキーボード入力制御に転用した、テキストベースのマクロ記述言語です。「時間軸に沿った状態の変更と実行」を基本とし、簡潔な記述で複雑な入力シーケンス、タイミング、同時押しを制御します。

## 2. 基本構文ルール

### 2.1 読みやすさと入力の区別
- **無効入力（無視）**: スペース、タブ、改行は読みやすさのための「空白」として扱われ、入力データとしては無視されます。
- **入力用スペース**: 明示的にスペースを入力する場合は `\s` を使用します。
- **特殊キー**: `[Enter]`, `[Tab]`, `[Esc]`, `[F1]` のように、角括弧 `[]` で囲んで記述します。

### 2.2 シェルエスケープとの共存
既存のシェルエスケープシーケンス（小文字）を保持しつつ、KML独自の制御コマンド（大文字）を割り当てます。

| 分類 | シーケンス | 意味・役割 | MML対応 |
| :--- | :--- | :--- | :--- |
| **標準入力** | `\n`, `\r`, `\t`, `\b` | 改行, 復帰, タブ, BS | - |
| **KML命令** | `\T[n]` | **Tempo**: 実行速度 (BPM) | T (Tempo) |
| | `\L[n]` | **Length**: デフォルトのキー押下時間 | L (Length) |
| | `\R[n]` | **Rest**: 待機時間 | R (Rest) |
| | `\O[n]` | **Octave**: キーレイヤーの切り替え | O (Octave) |
| **拡張入力** | `\\` | バックスラッシュリテラル | - |
| | `\'`, `\"` | 引用符リテラル | - |
| | `\s` | 半角スペースのリテラル | - |

## 3. 入力制御

### 3.1 キーと長さ
アルファベットや数字の直後に数字を書くことで、そのキーの持続時間を指定できます。
- **例**: `A4 B8 C`
  - Aを4分音符分、Bを8分音符分、Cをデフォルト長（`\L`）で入力。

### 3.2 区間制御 (Modifier & Range)
バックスラッシュを伴う括弧 `\[` `\]` を使用して、特定の状態が持続する範囲を定義します。
- **例**: `\[Shift hello \]` → Shiftを押しながら「hello」と入力。
- **同時押し**: `\[Ctrl+Alt [Delete] \]` → Ctrl+Alt+Delを実行。

### 3.3 タイ (Tie)
- **例**: `A&4` → Aを4分音符の長さだけ押し続ける（次の音と結合）。

## 4. CLIでの利用
KMLはシェル文字列として渡されることを想定しています。シェルのメタ文字解釈を避けるため、**シングルクォートで囲んで実行**することを推奨します。

```bash
# 使用例
kml-exec '\T180 \[Shift Pass \] \s \R4 [Enter]'
```

## 5. 構造例：コピー＆ペースト

初期案の「一行コピー」を最新仕様で記述：
`\T180 \L0 \[Ctrl c \] \R8 [End] \n \[Ctrl v \]`

1. **`\T180`**: テンポ設定。
2. **`\[Ctrl c \]`**: コピー（範囲指定による同時押し）。
3. **`\R8`**: 処理待ちの待機。
4. **`[End] \n`**: 行末移動と改行。
5. **`\[Ctrl v \]`**: 貼り付け。

## 6. 実装ガイドライン

1. **パーサ**: `\` 起点のシーケンスを優先抽出。
2. **スケジューラ**: `\T` に基づき各イベント間のミリ秒を算出。
3. **ドライバ**: OSのキーボードAPI（`SendInput`等）やマイコンの `Keyboard` ライブラリへ橋渡し。
