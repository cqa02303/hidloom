### 2. 座標変換式

#### 物理座標 → 1u座標変換
```
基板右上原点: (337.17, 79.12) mm
1u = 19.05mm

変換式:
x_1u = (337.17 - x_mm) / 19.05
y_1u = (y_mm - 79.12) / 19.05
```

### 3. ROW/COL GPIO対応

現在の `matrixd` はチャーリープレックス scan なので、ROW線 / COL線に同じ GPIO 配列を使います。

#### scan line GPIO

- LINE1: GPIO13 (Pin 33)
- LINE2: GPIO26 (Pin 37)
- LINE3: GPIO6 (Pin 31)
- LINE4: GPIO5 (Pin 29)
- LINE5: GPIO4 (Pin 7)
- LINE6: GPIO25 (Pin 22)
- LINE7: GPIO24 (Pin 18)
- LINE8: GPIO22 (Pin 15)
- LINE9: GPIO23 (Pin 16)
- LINE10: GPIO27 (Pin 13)

`config/default/matrixd.json`:

```json
"row_gpios": [13, 26, 6, 5, 4, 25, 24, 22, 23, 27],
"col_gpios": [13, 26, 6, 5, 4, 25, 24, 22, 23, 27]
```

SPI0 (`GPIO07`-`GPIO11`) は PAW mouse sensor 用に空けます。matrix scan では使いません。

### 4. チャーリープレックス スキャン手順

```python
# 疑似コード例
for row in range(10):
    # 指定ROWをHIGH、他をLOW
    set_row_high(row)
    microsecond_delay(1)  # セットリング時間

    # 全COLを読み取り
    for col in range(10):
        key_state = read_col(col)
        if key_state:
            register_keypress(row, col)

    set_row_low(row)
```
